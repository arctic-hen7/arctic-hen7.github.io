About Me

Hey there! I'm arctic_hen7, a fullstack web developer with a passion for productivity and anything cool! I mostly work with Rust, but I have an extensive background in technologies like ReactJS, NextJS, and Gatsby. Now, I aim to make technologies like these even more performant and astonishing through the power of Rust and WASM.


This site is my personal space on the internet to blog about everything web development, productivity, philosophy, and really anything! I update this frequently with blog posts about stuff I'm working on and problems I'm solving, with the hope that my posts will help both future me and someone else!


If you're interested, this site is built with Eleventy and Nunjucks, and is styled with TailwindCSS! The fonts used are Comfortaa and Raleway. Comments and reactions use Giscus, which depends on GitHub discussions! All the code is on GitHub, so please open an issue if there's anything in particular you'd like to see here!


If you want to contact me about collaboration or something interesting, drop me an email and let me know!
